Calculator

A Calculator app, that works for basic mathematical operations, where the results are going to be saved chronologically into a database, so a history can be created and visualized on the page.

Features

  • Basic mathematical operations with two numbers
  • Input validation
  • Show a list of previously done calculations

Used technologies

  • Grails Framework

Approach

First of all, I started by building the base mvc structure of the project by using in course material as template. I got a quite amount of errors in the beginning but managed to fix them. After the setup was done, I started creating the logic for the mathematical calculations and continued by integrating this into the view. Then it took me a big load of time to create a proper styling with css, so it looks decent.
